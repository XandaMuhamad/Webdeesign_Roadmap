    });
}

  • String
  • Number
  • Boolean
  • Object
  • Undefined

Functions

Functions are blocks of code designed to perform a particular task. They are executed when "called" (invoked).

function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Alice")); // Hello, Alice
Close

Control Structures

Control structures like loops and conditionals control the flow of execution in a program. Examples include:

if (age > 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}
Close

Arrays

Arrays are used to store multiple values in a single variable.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Apple
Close

Objects

Objects in JavaScript are used to store collections of data and more complex entities.

let person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};
console.log(person.fullName()); // John Doe
Close

DOM Manipulation

DOM manipulation allows you to interact with and modify the structure of a web page. Common methods include:

document.getElementById("myElement").innerHTML = "Hello, World!";
document.querySelector(".myClass").style.color = "blue";

Event Handling

Event handling allows you to respond to user interactions. Common events include:

document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

Asynchronous JavaScript

Asynchronous programming allows you to perform tasks without blocking the main thread. Common techniques include:

setTimeout(function() {
  console.log("This runs after 2 seconds");
}, 2000);

fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data));
