Basic HTML Tags

          <div class="container">
            <h1 id="title">Hello World</h1>
            <p class="description">This is a paragraph</p>
          </div>

Links

Hyperlinks are created using the <a> tag. They allow you to link to other web pages or resources.

<a  href="https://example.com">Visit Example</a>
Close

Lists

HTML provides two types of lists: ordered lists (<ol>) and unordered lists (<ul>).

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>
Close

Images

Images can be embedded using the <img> tag. You can also add attributes like alt for accessibility.

<img src="image.jpg" alt="Description">
Close

Attributes

Attributes provide additional information about HTML elements. Common attributes include id, class, and style.

<div id="uniqueId" class="className" style="color: red;">Content</div>
Close

HTML Forms

HTML forms are used to collect user inputs and send data to a server. Important form elements include:

  • <input> – Used for various input types such as text, password, email, etc.
  • <textarea> – Allows multi-line text input.
  • <button> – Used for submitting forms.
<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <textarea id="message" name="message"></textarea>
  <button type="submit">Submit</button>
</form>
Close

HTML Tables

HTML tables are used to display tabular data. Key elements include:

  • <table> – Defines the table.
  • <tr> – Defines a table row.
  • <td> – Defines a table cell.
  • <th> – Defines a table header cell.
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
  </tr>
</table>
Close

HTML Media

HTML allows you to embed various types of media, such as images, videos, and audio. Key elements include:

  • <img> – Embeds an image.
  • <video> – Embeds a video.
  • <audio> – Embeds an audio file.
<img src="image.jpg" alt="Description">
<video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>
Close

HTML Semantics

Explore the importance of semantic HTML for accessibility and SEO. Semantic elements clearly describe their meaning in a human- and machine-readable way.

  • <header> – Defines a header for a document or section.
  • <nav> – Defines a set of navigation links.
  • <article> – Defines an independent, self-contained content.
  • <section> – Defines a section in a document.
  • <footer> – Defines a footer for a document or section.
<header>This is a header</header>
<nav>This is a navigation</nav>
<article>This is an article</article>
<section>This is a section</section>
<footer>This is a footer</footer>
Close
