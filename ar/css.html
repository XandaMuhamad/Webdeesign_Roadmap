المحددات

تُستخدم محددات CSS لتحديد العناصر التي تريد تنسيقها. تشمل المحددات الشائعة:

  • محدد العنصر – يحدد جميع العناصر من نوع معين.
  • محدد الفئة – يحدد جميع العناصر التي تحتوي على فئة معينة.
  • محدد المعرف – يحدد عنصرًا واحدًا بمعرف معين.
p {
  color: blue;
}
.class-name {
  font-size: 16px;
}
#id-name {
  margin: 10px;
}
إغلاق

نموذج الصندوق

يصف نموذج الصندوق في CSS الصناديق المستطيلة التي يتم إنشاؤها للعناصر في شجرة المستند ويتكون من:

  • المحتوى – المحتوى الفعلي للصندوق، حيث تظهر النصوص والصور.
  • الحشو – يفرغ منطقة حول المحتوى. الحشو شفاف.
  • الحدود – حد يحيط بالحشو (إن وجد) والمحتوى.
  • الهامش – يفرغ منطقة خارج الحدود. الهامش شفاف.
div {
  padding: 10px;
  border: 1px solid black;
  margin: 20px;
}
إغلاق

العرض

تحدد خاصية العرض سلوك العرض (نوع صندوق العرض) للعنصر. تشمل القيم الشائعة:

  • block – يتم عرض العنصر كعنصر على مستوى الكتلة.
  • inline – يتم عرض العنصر كعنصر على مستوى السطر.
  • flex – يتم عرض العنصر كحاوية مرنة على مستوى الكتلة.
.block-element {
  display: block;
}
.inline-element {
  display: inline;
}
.flex-container {
  display: flex;
}
إغلاق

الوحدات

لدى CSS عدة وحدات مختلفة للتعبير عن الطول. تشمل الوحدات الشائعة:

  • px – البكسل.
  • % – النسبة المئوية.
  • em – نسبة إلى حجم الخط للعنصر.
  • rem – نسبة إلى حجم الخط للعنصر الجذري.
div {
  width: 100px;
  height: 50%;
  padding: 1em;
  margin: 2rem;
}
إغلاق

Flexbox

يُسهل نموذج تخطيط الصندوق المرن في CSS تصميم هيكل تخطيط مرن مستجيب دون استخدام التطفو أو التمركز.

  • display – يحدد حاوية مرنة.
  • flex-direction – يحدد اتجاه العناصر المرنة.
  • justify-content – يحاذي العناصر المرنة على طول المحور الرئيسي.
  • align-items – يحاذي العناصر المرنة على طول المحور العرضي.
.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}
إغلاق

الشبكة

يوفر نموذج تخطيط الشبكة في CSS نظام تخطيط قائم على الشبكة، مع صفوف وأعمدة، مما يسهل تصميم صفحات الويب دون الحاجة إلى استخدام التطفو والتمركز.

  • display – يحدد حاوية الشبكة.
  • grid-template-columns – يحدد أعمدة الشبكة.
  • grid-template-rows – يحدد صفوف الشبكة.
  • grid-gap – يحدد الفجوة بين عناصر الشبكة.
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  grid-gap: 10px;
}
إغلاق

الرسوم المتحركة في CSS

تسمح الرسوم المتحركة في CSS بتحريك الانتقالات بين أنماط CSS. تشمل الخصائص الرئيسية:

  • animation-name – يحدد اسم الرسوم المتحركة @keyframes.
  • animation-duration – يحدد مدة الرسوم المتحركة.
  • animation-timing-function – يحدد منحنى سرعة الرسوم المتحركة.
@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

div {
  animation-name: example;
  animation-duration: 4s;
  animation-timing-function: ease-in-out;
}
إغلاق

الألوان

تسمح CSS بتعيين الألوان للنصوص والخلفيات والحدود والعناصر الأخرى. تشمل الطرق الشائعة لتحديد الألوان:

  • السداسي العشري – مثل #ff0000 للأحمر.
  • RGB – مثل rgb(255, 0, 0) للأحمر.
  • RGBA – مثل rgba(255, 0, 0, 0.5) للأحمر مع 50% شفافية.
  • الألوان المسماة – مثل red.
p {
  color: #ff0000;
  background-color: rgba(0, 0, 255, 0.1);
}
إغلاق

الطباعة

تشمل الطباعة في CSS تنسيق النصوص. تشمل الخصائص الرئيسية:

  • font-family – يحدد الخط.
  • font-size – يحدد حجم النص.
  • font-weight – يحدد سمك النص.
  • line-height – يحدد المسافة بين سطور النص.
p {
  font-family: 'Arial, sans-serif';
  font-size: 16px;
  font-weight: bold;
  line-height: 1.5;
}
إغلاق
