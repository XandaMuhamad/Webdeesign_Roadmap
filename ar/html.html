دلالات HTML

استكشاف أهمية HTML الدلالية لتحسين الوصول وتحسين محركات البحث.

العلامات الأساسية في HTML

تُستخدم علامات HTML لتنظيم محتوى الويب. بعض العلامات الأساسية تشمل:

  • <h1>هذا عنوان</h1>
  • <p>هذه فقرة.</p>
  • <a href="https://example.com">هذا رابط</a>
  • <img src="image.jpg" alt="وصف">
<h1>هذا عنوان</h1>
<p>هذه فقرة.</p>
<a href="https://example.com">هذا رابط</a>
<img src="image.jpg" alt="وصف">
إغلاق

الروابط

تُستخدم الروابط لإنشاء روابط بين صفحات الويب.

<a href="https://example.com">زيارة Example</a>
إغلاق

القوائم

يوفر HTML نوعين من القوائم: القوائم المرتبة (<ol>) والقوائم غير المرتبة (<ul>).

<ul>
  <li>البند 1</li>
  <li>البند 2</li>
</ul>

<ol>
  <li>البند الأول</li>
  <li>البند الثاني</li>
</ol>
إغلاق

الصور

يمكن تضمين الصور باستخدام علامة <img>. يمكنك أيضًا إضافة سمات مثل <alt> لتحسين الوصول.

<img src="image.jpg" alt="وصف">
إغلاق

السمات

توفر السمات معلومات إضافية حول عناصر HTML. تشمل السمات الشائعة <id> و<class> و<style>.

<div id="uniqueId" class="className" style="color: red;">المحتوى</div>
إغلاق

النماذج

تُستخدم نماذج HTML لجمع مدخلات المستخدم وإرسال البيانات إلى الخادم. تشمل عناصر النموذج المهمة:

  • <label> و<input>
  • <textarea>
  • <button>
<form>
  <label for="name">الاسم:</label>
  <input type="text" id="name" name="name">
  <textarea id="message" name="message"></textarea>
  <button type="submit">إرسال</button>
</form>
إغلاق

الجداول

تُستخدم الجداول في HTML لعرض البيانات في شكل جدولي. تشمل العناصر الرئيسية:

  • <table>
  • <tr>
  • <th> و<td>
<table>
  <tr>
    <th>الاسم</th>
    <th>العمر</th>
  </tr>
  <tr>
    <td>جون</td>
    <td>30</td>
  </tr>
</table>
إغلاق

الوسائط

يسمح HTML بتضمين أنواع مختلفة من الوسائط، مثل الصور والفيديوهات والصوت. تشمل العناصر الرئيسية:

  • <img>
  • <video>
  • <audio>
<img src="image.jpg" alt="وصف">
<video controls>
  <source src="video.mp4" type="video/mp4">
  متصفحك لا يدعم علامة الفيديو.
</video>
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  متصفحك لا يدعم عنصر الصوت.
</audio>
إغلاق
