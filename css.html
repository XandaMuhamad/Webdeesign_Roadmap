هەڵبژێرەکان و تایبەتمەندییەکان

فێربە چۆن CSS هەڵدەبژێرێت و شێواز دەدات بە توخمەکانی HTML.

مۆدێلی بۆکس

تێبگە لە مۆدێلی بۆکسی CSS و چۆنیەتی کارکردنی پەراوێز، سنوور و پادینگ.

تایبەتمەندییەکانی پیشاندان

شارەزای تایبەتمەندییە جیاوازەکانی پیشاندان ببە لە CSS، وەک block، inline و flexbox.

یەکەکانی CSS

فێری یەکە ڕەها و ڕێژەییەکان ببە لە CSS، وەک px، % و em.

فلێکس بۆکس

شارەزای مۆدێلی لەیاوتی فلێکس بۆکس ببە بۆ دیزاینی وەڵامدەر.

گرید CSS

فێری لەیاوتی گریدی CSS ببە بۆ دروستکردنی لەیاوتی ئاڵۆز.

ئەنیمەیشنەکانی CSS

فێربە چۆن ئەنیمەیشن و گواستنەوە دروست دەکەیت لە CSS.

ڕەنگەکان

فێربە چۆن ڕەنگەکان بەکاردەهێنیت لە CSS بۆ ڕازاندنەوەی پەڕەکانی وێب.

تایپۆگرافی

تێبگە لە چۆنیەتی ڕازاندنەوەی دەق بە بەکارهێنانی تایبەتمەندییەکانی تایپۆگرافی CSS.

هەڵبژێرەکان و تایبەتمەندییەکان

هەڵبژێرەکانی CSS نەخشەن کە بەکاردێن بۆ هەڵبژاردنی ئەو توخمانەی دەتەوێت شێوازیان پێ بدەیت. هەڵبژێرە باوەکان ئەمانەن:

  • هەڵبژێری توخم – توخمەکان بە ناوی تاگەکانیان دەستنیشان دەکات (بۆ نموونە، p)
  • هەڵبژێری کڵاس – توخمەکان بە کڵاسەکانیان دەستنیشان دەکات (بۆ نموونە، .class-name)
  • هەڵبژێری ID – توخمەکان بە ID-یەکانیان دەستنیشان دەکات (بۆ نموونە، #id-name)

کۆد:

p {
  color: blue;
}

.class-name {
  font-size: 16px;
}

#id-name {
  margin: 10px;
}

مۆدێلی بۆکس

مۆدێلی بۆکسی CSS گرنگە بۆ تێگەیشتن لە لەیاوت و دیزاین. پێکهاتووە لە:

  • ناوەڕۆک – ناوەڕۆکی ڕاستەقینەی بۆکسەکە، وەک دە یان وێنە
  • پادینگ – ناوچەیەک پاک دەکاتەوە لە دەوری ناوەڕۆک، لە ناو سنوورەکە
  • سنوور – سنوورێک کە دەچێتە دەوری پادینگ و ناوەڕۆک
  • مارجن – ناوچەیەک پاک دەکاتەوە لە دەرەوەی سنوور، توخمەکە لە ئەوانی تر جیا دەکاتەوە

کۆد:

div {
  padding: 10px;
  border: 1px solid black;
  margin: 20px;
  background-color: #f0f0f0;
}

دەرئەنجام:

ئەمە بۆکسێکە کە پادینگ، سنوور و مارجنی هەیە

تایبەتمەندییەکانی پیشاندان

تایبەتمەندییەکانی پیشاندانی CSS دیاری دەکەن کە چۆن توخمەکان لەسەر شاشە پیشان دەدرێن. بەهاکانی باو ئەمانەن:

  • block – توخمەکە هەموو پانییەکی بەردەست دەگرێت، لە هێڵێکی نوێوە دەست پێدەکات
  • inline – توخمەکە تەنها ئەوەندە پانی دەگرێت کە ناوەڕۆکەکەی پێویستیەتی، بەبێ دەستپێکردن لە هێڵێکی نوێ
  • flex – لەیاوتی بۆکسی نەرم چالاک دەکات، گونجاوە بۆ دیزاینی وەڵامدەر

کۆد:

.block-element {
  display: block;
}

.inline-element {
  display: inline;
}

.flex-container {
  display: flex;
}

یەکەکانی CSS

یەکەکانی CSS درێژی، پانی و دووری لە شێوازدا دیاری دەکەن. یەکە باوەکان ئەمانەن:

  • px – پیکسڵ، یەکەیەکی ڕەها
  • % – پەیوەندیدارە بە توخمی باوک
  • em – پەیوەندیدارە بە قەبارەی فۆنتی توخمەکە
  • rem – پەیوەندیدارە بە قەبارەی فۆنتی توخمی ڕەگ

کۆد:

div {
  width: 100px;
  height: 50%;
  padding: 1em;
  margin: 2rem;
}

بۆکسێک بە یەکە جیاوازەکان

فلێکس بۆکس

فلێکس بۆکس مۆدێلێکی لەیاوتە کە ڕێگە دەدات بە توخمەکان ڕێک بخرێن و بۆشایی لە ناو کۆنتەینەرێکدا دابەش بکرێت. تایبەتمەندییە سەرەکییەکان ئەمانەن:

  • flex-direction – ئاراستەی توخمەکانی فلێکس دیاری دەکات
  • justify-content – توخمەکان لە تەوەری سەرەکی ڕێک دەخات
  • align-items – توخمەکان لە تەوەری لاوەکی ڕێک دەخات

کۆد:

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

تۆڕی CSS

لەیاوتی تۆڕی CSS سیستەمێکی دوو-ڕەهەندییە بۆ وێب. چەمکە سەرەکییەکان ئەمانەن:

  • grid-template-columns – ستوونەکانی تۆڕەکە دیاری دەکات
  • grid-template-rows – ڕیزەکانی تۆڕەکە دیاری دەکات
  • grid-gap – بۆشایی نێوان ڕیز و ستوونەکان دیاری دەکات

کۆد:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  grid-gap: 10px;
}

جووڵەکانی CSS

جووڵەکانی CSS ڕێگەت پێدەدات جووڵە دروست بکەیت لە نێوان شێوازەکانی CSS. تایبەتمەندییە سەرەکییەکان ئەمانەن:

  • animation-name – ناوی جووڵەکە دیاری دەکات لە @keyframes
  • animation-duration – ماوەی جووڵەکە دیاری دەکات
  • animation-timing-function – هێڵکاری خێرایی جووڵەکە دیاری دەکات

کۆد:

@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}

div {
  animation-name: example;
  animation-duration: 4s;
  animation-timing-function: ease-in-out;
}

بۆکسێکی جووڵاو

ڕەنگەکان

CSS ڕێگەت پێدەدات ڕەنگ بدەیت بە دەق، پاشبنەما، سنوور و توخمەکانی تر. شێوازە باوەکانی دیاریکردنی ڕەنگ ئەمانەن:

  • شازدەیی – بۆ نموونە، #ff0000 بۆ سوور
  • RGB – بۆ نموونە، rgb(255, 0, 0) بۆ سوور
  • RGBA – بۆ نموونە، rgba(255, 0, 0, 0.5) بۆ سووری نیوە ڕوون
  • ناوی ڕەنگەکان – بۆ نموونە، red

کۆد:

p {
  color: #ff0000;
  background-color: rgba(0, 0, 255, 0.1);
}

دەرئەنجام:

ئەمە دەقێکە بە ڕەنگی سوور و پاشبنەمایەکی شینی کاڵ

تایپۆگرافی

تایپۆگرافی لە CSS پەیوەندی بە شێوازدانی دەقەوە هێڵکاری دەکات. تایبەتمەندییە سەرەکییەکان ئەمانەن:

  • font-family – جۆری فۆنت دیاری دەکات
  • font-size – قەبارەی دەق دیاری دەکات
  • font-weight – ئەستووری دەق دیاری دەکات
  • line-height – بۆشایی نێوان هێڵەکان دیاری دەکات

کۆد:

p {
  font-family: 'Arial, sans-serif';
  font-size: 16px;
  font-weight: bold;
  line-height: 1.5;
}

ئەمە نموونەیەکە لە دەقێک کە شێوازی تایپۆگرافی پێدراوە. بۆ نیشاندانی line-height، با چەند دێڕێک زیاد بکەین.
