تاگە بنەڕەتییەکان

فێربە دەربارەی تاگە بنەڕەتییەکانی HTML کە پێکهاتەی ناوەڕۆک لە وێب دروست دەکەن.

بەستەرەکان

فێربە چۆن بەستەر دروست بکەیت بۆ پەیوەندی کردنی لاپەڕەکانی وێب.

لیستەکان

تێگەیشتن لە چۆنیەتی دروستکردنی لیستی ڕێکخراو و ناڕێکخراو.

وێنەکان

فێربە چۆن وێنە دابنێیت و ستایلی بکەیت لە HTML.

تایبەتمەندییەکان

تێگەیشتن لە بەکارهێنانی تایبەتمەندییەکانی HTML بۆ زیادکردنی زانیاری زیاتر بۆ توخمەکان.

فۆرمەکان

تێگەیشتن لە چۆنیەتی دروستکردنی فۆرم بۆ وەرگرتنی داتا لە بەکارهێنەر لە HTML.

خشتەکان

فێربە چۆن خشتە دروست بکەیت و ستایلی بکەیت لە HTML.

میدیا

تاگە بنەڕەتییەکانی HTML

تاگەکانی HTML بەکاردێن بۆ پێکهاتەی ناوەڕۆک لە وێب. هەندێک لە تاگە گرنگەکان:

  • <h1> تا <h6> – سەردێڕەکان، کە <h1> بەرزترین ئاستە.
  • <p> – تاگی پەرەگراف بۆ دەقی ناوەڕۆک.
  • <a> – تاگی لەنگەر بۆ دروستکردنی بەستەر.
  • <img> – تاگی وێنە بۆ دانانی وێنەکان.

کۆد:

<h1>سەردێڕی سەرەکی</h1>
<h2>سەردێڕی لاوەکی</h2>
<p>ئەمە پەرەگرافێکە.</p>
<a href="https://example.com">بەستەر</a>

فۆرمەکانی HTML

فۆرمەکانی HTML بەکاردێن بۆ کۆکردنەوەی داخڵکراوەکانی بەکارهێنەر و ناردنی داتا بۆ ڕاژە. توخمە گرنگەکانی فۆرم ئەمانەن:

  • <input> – بەکاردێت بۆ جۆرەکانی داخڵکردن وەک دەق، وشەی نهێنی، ئیمەیڵ و هتد.
  • <textarea> – ڕێگە دەدات بە داخڵکردنی دەقی فرە-هێڵ.
  • <button> – بەکاردێت بۆ ناردنی فۆرمەکان.

کۆد:

<form>
  <label for="name">ناو:</label>
  <input type="text" id="name" name="name">
  <textarea id="message" name="message"></textarea>
  <button type="submit">ناردن</button>
</form>

واتاسازی HTML

HTML ی واتایی تاگەکان بەکاردێنێت کە واتایان هەیە و یارمەتی پێکهاتەی ناوەڕۆک دەدەن بە شێوەیەکی کاریگەرتر. تاگە واتاییە سەرەکییەکان:

  • <header> – نوێنەرایەتی ناوەڕۆکی سەرەتایی یان کۆمەڵێک بەستەری ڕێنیشاندەر دەکات.
  • <article> – پێکهاتەیەکی سەربەخۆیە لە پەڕەی وێب.
  • <footer> – نوێنەرایەتی پێپەڕەی بەش یان لاپەڕە دەکات.
  • <section> – نوێنەرایەتی گروپێکی بابەتی ناوەڕۆک دەکات.

کۆد:

<header>
  <h1>بەخێربێیت بۆ ماڵپەڕەکەم</h1>
</header>

<article>
  <h2>ناونیشانی وتار</h2>
  <p>ئەمە وتارێکە.</p>
</article>

<footer>
  <p>© ٢٠٢٤ ماڵپەڕەکەم</p>
</footer>

ئەمە وتارێکە.

خشتەکانی HTML

خشتەکانی HTML بەکاردێن بۆ پیشاندانی داتای خشتەیی. توخمە سەرەکییەکان:

  • <table> – خشتە پێناسە دەکات.
  • <tr> – ڕیزی خشتە پێناسە دەکات.
  • <td> – خانەی خشتە پێناسە دەکات.
  • <th> – خانەی سەرەوەی خشتە پێناسە دەکات.

کۆد:

<table>
  <tr>
    <th>ناو</th>
    <th>تەمەن</th>
  </tr>
  <tr>
    <td>ئازاد</td>
    <td>٣٠</td>
  </tr>
</table>

دەرئەنجام:

ناو تەمەن
ئازاد ٣٠

میدیای HTML

HTML ڕێگە دەدات بە تێخستنی جۆرەها میدیا، وەک وێنە، ڤیدیۆ و دەنگ. توخمە سەرکییەکان:

  • <img> – وێنە تێدەخات.
  • <video> – ڤیدیۆ تێدەخات.
  • <audio> – فایلی دەنگ�� تێدەخات.

کۆد:

<img src="image.jpg" alt="وەسف"
  width="300">

<video controls>
  <source src="video.mp4" type="video/mp4">
  وێبگەڕەکەت پشتگیری تاگی ڤیدیۆ ناکات.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  وێبگەڕەکەت پشتگیری توخمی دەنگ ناکات.
</audio>

دەرئەنجام:

📝 تێبینی: ئەم نموونانە تەنها بۆ پیشاندانن

لیستەکان

HTML دوو جۆر لیست دابین دەکات: لیستی ڕێکخراو (<ol>) و لیستی ناڕێکخراو (<ul>).

کۆد:

<ul>
  <li>بڕگەی ١</li>
  <li>بڕگەی ٢</li>
</ul>

<ol>
  <li>بڕگەی یەکەم</li>
  <li>بڕگەی دووەم</li>
</ol>

دەرئەنجام:

  • بڕگەی ١
  • بڕگەی ٢
  1. بڕگەی یەکەم
  2. بڕگەی دووەم

وێنەکان

وێنەکان دەکرێت بە بەکارهێنانی تاگی <img> تێبخرێن. هەروەها دەتوانیت تایبەتمەندی وەک alt بۆ دەستگەیشتن زیاد بکەیت.

کۆد:

<img 
  src="image.jpg" 
  alt="وەسف"
  width="300">

دەرئەنجام:

تێبینی: ئەم وێنەیە تەنها نموونەیە

تایبەتمەندییەکان

تایبەتمەندییەکان زانیاری زیاتر دەربارەی توخمەکانی HTML دابین دەکەن. تایبەتمەندییە باوەکان id، class، و style لەخۆدەگرن.

کۆد:

<div 
  id="ناسنامەی_تایبەت" 
  class="ناوی_پۆل" 
  style="color: red; padding: 10px; border: 1px solid #ddd;"
>
  ناوەڕۆک
</div>

دەرئەنجام:

ناوەڕۆک

تێبینی:

  • id: ناسنامەیەکی تایبەت بۆ توخمەکە
  • class: پۆلێک بۆ شێوازدان
  • style: شێوازی ڕاستەوخۆ
