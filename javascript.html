گۆڕاوەکان و جۆرەکانی داتا

گۆڕاوەکان لە جاڤاسکریپت بەکاردێن بۆ خەزنکردنی زانیاری.

نموونەی گۆڕاوەکان:

let name = "ئازاد";
let age = 30;
let isStudent = true;
let person = { 
    name: "ئازاد", 
    age: 30 
};

دەرئەنجام:

> console.log(name);
"ئازاد"
> console.log(age);
30
> console.log(isStudent);
true

جۆرەکانی داتا:

  • string - بۆ دەق (وەک "ئازاد")
  • number - بۆ ژمارە (وەک ٣٠)
  • boolean - بۆ ڕاست/هەڵە
  • object - بۆ کۆمەڵێک خاسیەت
  • array - بۆ لیستی بەهاکان

فەنکشنەکان

فەنکشنەکان کۆدێکن کە دەتوانرێت دووبارە بەکاربهێنرێنەوە و یارمەتیدەرن بۆ ڕێکخستنی کۆد.

فەنکشنی سادە:

function sayHello() {
    console.log("سڵاو، جیهان!");
}
sayHello(); // Output: سڵاو، جیهان!

دەرئەنجام:

> sayHello();
سڵاو، جیهان!

فەنکشن لەگەڵ پارامیتەر:

function addNumbers(a, b) {
    return a + b;
}
console.log(addNumbers(2, 3)); // Output: 5

دەرئەنجام:

> addNumbers(2, 3);
5

دەستکاری DOM

دەستکاری DOM ڕێگەت پێدەدات کارلێک و گۆڕانکاری لە پێکهاتەی پەڕەی وێب بکەیت.

گۆڕینی ناوەڕۆک:

// گۆڕینی ناوەڕۆکی توخمێک
    document.getElementById("myElement").innerHTML = "سڵاو، جیهان!";
    
    // گۆڕینی ستایل
    document.querySelector(".myClass").style.color = "blue";
    
    // وەرگرتنی ڕووداو
    document.getElementById("myButton").addEventListener("click", function() {
        alert("دوگمەکە کلیک کرا!");
    });

میسۆدە باوەکان:

  • getElementById() - دۆزینەوەی توخمێک بە ID
  • querySelector() - دۆزینەوەی توخمێک بە CSS selector
  • addEventListener() - زیادکردنی ڕووداو

تێبینییەکان:

  • DOM واتە Document Object Model - مۆدێلی پێکهاتەی دۆکیومێنت
  • هەموو توخمەکانی HTML دەتوانرێت لە ڕێگەی DOM ەوە دەستکاری بکرێن
  • دەتوانیت توخمەکان زیاد بکەیت، بسڕیتەوە، یان دەستکارییان بکەیت
  • ستایل و کڵاسەکانیش دەتوانرێت بگۆڕدرێن

مامەڵەکردن لەگەڵ ڕووداوەکان

مامەڵەکردن لەگەڵ ڕووداوەکان ڕێگەت پێدەدات وەڵامی کارلێکی بەکارهێنەر بدەیتەوە.

نموونەی ڕووداوەکان:

// ڕووداوی کلیک
    document.getElementById("myButton").addEventListener("click", function() {
        alert("دوگمەکە کلیک کرا!");
    });
    
    // ڕووداوی گۆڕان
    document.querySelector("input").addEventListener("change", function(event) {
        console.log("نرخی نوێ:", event.target.value);
    });

ڕووداوە باوەکان:

  • click - کاتێک توخمێک کلیک دەکرێت
  • change - کاتێک نرخی فۆرمێک دەگۆڕدرێت
  • submit - کاتێک فۆرمێک سەبمیت دەکرێت
  • keyup - کاتێک کلیلێک بەردەدرێت
  • mouseover - کاتێک ماوس دێتە سەر توخمێک

تێبینییەکان:

  • ڕووداوەکان بەشێکی گرنگن لە وێب دەڤەڵۆپمێنت
  • دەتوانیت چەندین ڕووداو بۆ یەک توخم زیاد بکەیت
  • ڕووداوەکان دەتوانن زانیاری زیاتر لەسەر ڕووداوەکە بگەڕێننەوە
  • دەتوانیت ڕووداوەکان لابەریت بە بەکارهێنانی removeEventListener

جاڤاسکریپتی ناهاوکات

پرۆگرامکردنی ناهاوکات ڕێگەت پێدەدات ئەرکەکان جێبەجێ بکەیت بەبێ بلۆککردنی سەرەکی.

setTimeout:

setTimeout(function() {
      console.log("ئەمە دوای ٢ چرکە جێبەجێ دەبێت");
  }, 2000);

Fetch API:

fetch("https://api.example.com/data")
      .then(response => response.json())
      .then(data => console.log(data));

تەکنیکە باوەکان:

  • setTimeout() - جێبەجێکردنی کۆد دوای ماوەیەکی دیاریکراو
  • fetch() - داواکردنی داتا لە سێرڤەر
  • Promise - بەڵێنی جێبەجێکردنی کۆد لە داهاتوودا
  • async/await - ڕێگایەکی ئاسانتر بۆ مامەڵەکردن لەگەڵ کۆدی ناهاوکات

تێبینییەکان:

  • کۆدی ناهاوکات یارمەتیدەرە بۆ باشترکردنی کارایی بەرنامەکە
  • بەکارهێنانی Promise باشترە لە callback
  • هەمیشە هەڵەکان بگرە (catch) لە کۆدی ناهاوکات
  • async/await کۆدەکە خوێندنەوەی ئاسانتر دەکات

ئۆبجێکتەکان

ئۆبجێکتەکان لە جاڤاسکریپت بەکاردێن بۆ خەزنکردنی کۆمەڵێک داتا و شتی ئاڵۆزتر.

نموونەی ئۆبجێکت:

let کەس = {
      ناو: "ئازاد",
      تەمەن: 30,
      ناویتەواو: function() {
          return this.ناو;
      }
  };
  
  console.log(کەس.ناویتەواو()); // ئازاد

تایبەتمەندییەکانی ئۆبجێکت:

  • properties - خاسیەتەکانی ئۆبجێکت (وەک ناو و تەمەن)
  • methods - فەنکشنەکانی ناو ئۆبجێکت
  • this - ئاماژە بۆ ئۆبجێکتی ئێستا

تێبینییەکان:

  • ئۆبجێکتەکان گرنگترین بەشی جاڤاسکریپتن
  • دەتوانیت خاسیەت و میسۆد زیاد و کەم بکەیت
  • ئۆبجێکتەکان دەتوانن لە یەکتر میرات وەربگرن
  • JSON فۆرماتێکی تایبەتە بۆ ئۆبجێکتەکان
داخستن

ئەڕەیەکان

ئەڕەیەکان بەکاردێن بۆ خەزنکردنی چەند نرخێک لە یەک گۆڕاودا.

دروستکردن و دەستگەیشتن:

let میوەکان = ["سێو", "مۆز", "گێلاس"];
console.log(میوەکان[0]); // سێو

شێوازەکانی ئەڕەی:

// زیادکردنی بەها
میوەکان.push("پرتەقاڵ");

// پیشاندانی هەموو بەهاکان
میوەکان.forEach(میوە => {
    console.log(میوە);
});

میسۆدە باوەکان:

  • push() - زیادکردنی بەها بۆ کۆتایی
  • pop() - لابردنی دوا بەها
  • forEach() - پیشاندانی هەموو بەهاکان
  • map() - گۆڕینی هەموو بەهاکان
  • filter() - فلتەرکردنی بەهاکان

تێبینییەکان:

  • ئەڕەیەکان لە ٠ەوە دەست پێدەکەن
  • دەتوانیت هەر جۆرە داتایەک لە ئەڕەی خەزن بکەیت
  • ئەڕەیەکان قەبارەیان دینامیکییە
  • زۆر میسۆدی بەسوود هەیە بۆ مامەڵەکردن لەگەڵ ئەڕەی

هەڵەدۆزینەوە

هەڵەدۆزینەوە پرۆسەی دۆزینەوە و چاککردنی هەڵەکانە لە کۆدەکەتدا.

تەکنیکەکانی هەڵەدۆزینەوە:

// پەیامی هەڵەدۆزینەوە
console.log("پەیامی هەڵەدۆزینەوە");

// وەستانی جێبەجێکردن
debugger;

// نیشاندانی زانیاری زیاتر
console.info("زانیاری وردەکاری");
console.error("هەڵەیەک ڕوویدا!");

ئامرازەکانی هەڵەدۆزینەوە:

  • console.log() - پیشاندانی زانیاری
  • debugger - وەستانی کۆد
  • console.error() - پیشاندانی هەڵە
  • try/catch - گرتنی هەڵەکان

تێبینییەکان:

  • هەمیشە هەڵەکان بگرە لە کۆدەکەتدا
  • DevTools یارمەتیدەرێکی باشە بۆ هەڵەدۆزینەوە
  • console.log باشترین هاوڕێی پرۆگرامەرە
  • breakpoints یارمەتیت دەدەن بۆ دۆزینەوەی کێشەکان
